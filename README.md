Book Search & Rating System
A command-line book tracking application.
The system lets a user search a book database, log books they've read, rate and review them, and maintain a personal to-read list — all powered by custom-implemented data structures.

bookshelf/
├── main.py          # Entry point — top-level CLI menu
├── book.py          # Book data class
├── bst.py           # Binary Search Tree (core data structure)
├── stack.py         # Stack via linked list (core data structure)
├── queue.py         # Queue via linked list (core data structure)
├── search.py        # Search & filter logic (uses BST)
